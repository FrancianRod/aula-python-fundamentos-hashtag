🚀 Sistema de Performance B2B: Automação de Bônus com Python
Este projeto consiste em um Motor de Decisão Automatizado projetado para o cenário B2B, focado em transformar regras de negócio complexas em um fluxo de trabalho eficiente, escalável e resiliente.

📋 Sobre o Projeto
A finalidade deste sistema é automatizar a política de premiação e incentivos de uma operação comercial. O script atua como uma ferramenta estratégica que:

Classifica consultores em categorias (Ouro, Prata ou Bronze) com base no faturamento atingido.

Calcula o valor exato do bônus proporcional ao faturamento reportado.

Garante a Integridade do processo através de tratamento de exceções e validação de tipos de dados.

🛠️ Tecnologias e Conceitos Aplicados
O desenvolvimento abrange pilares fundamentais da programação voltada para automação de processos:

Gestão de Dados: Organização de constantes e variáveis dinâmicas.

Interatividade: Captura e processamento de informações via interface de linha de comando.

Lógica de Decisão: Construção de motores de regras utilizando estruturas condicionais encadeadas (if/elif/else).

Resiliência de Software: Implementação de blocos try/except para tratamento de erros de entrada e prevenção de falhas críticas (Runtime Errors).

💻 Fluxo de Funcionamento
O sistema opera em cinco etapas lógicas:

Configuração: Definição dos parâmetros de metas e alíquotas de premiação.

Captura: Coleta dinâmica do nome do consultor e faturamento total.

Validação (Casting): Conversão técnica de strings para tipos numéricos com tratamento de erros.

Processamento: Execução da lógica de classificação automática conforme a performance.

Entrega: Geração de feedback profissional com valores formatados e diretrizes de bonificação.
